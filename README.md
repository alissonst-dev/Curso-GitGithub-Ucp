# Git e GitHub na Prática

Projeto de extensão voltado ao ensino de Git, GitHub e desenvolvimento colaborativo, realizado na UCP – Faculdades do Centro do Paraná, com foco na utilização de ferramentas de controle de versão amplamente empregadas no mercado de tecnologia.

---

## Sobre o Projeto

O projeto tem como objetivo apresentar os conceitos fundamentais de controle de versão e colaboração em projetos de software utilizando Git e GitHub.

As atividades foram desenvolvidas em formato prático, permitindo que os participantes criassem, gerenciassem e colaborassem em repositórios reais, simulando fluxos de trabalho utilizados em equipes de desenvolvimento.

---

## Objetivos

* Compreender a importância do controle de versão.
* Conhecer os conceitos fundamentais do Git.
* Utilizar o GitHub para hospedagem e compartilhamento de projetos.
* Registrar alterações através de commits.
* Trabalhar com branches e integração de código.
* Utilizar Pull Requests para revisão e colaboração.
* Aplicar um fluxo colaborativo semelhante ao utilizado em equipes de desenvolvimento.

---

## Conteúdo Programático

### Encontro 1 — Fundamentos do Git e GitHub (4h)

* Introdução ao controle de versão
* O que são Git e GitHub
* Instalação e configuração do Git
* VS Code e terminal
* Criação de conta no GitHub
* Criação de repositórios
* Git Init
* Git Status
* Git Add
* Git Commit
* Git Log
* Git Push
* Arquivos ignorados com `.gitignore`
* Remoção de arquivos do rastreamento com `git rm --cached`
* Exercício prático individual

### Encontro 2 — Git Colaborativo (4h)

* Conceito de Branch
* Git Branch
* Git Checkout
* Git Switch
* Git Fetch
* Git Pull
* Git Merge
* Conflitos de Merge
* Pull Request (PR)
* Exclusão de Branches locais e remotas
* Fluxo colaborativo completo
* Desafio prático em equipe
